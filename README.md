# Frontend (React.js) Ecommerce Application

The application would be split into two parts: a front-end user interface built with React JS and Redux toolkit a back-end server built with JSON server . The database management would be handled by json 


## Features

- Product Catalog
- Shopping Cart
- Checkout Process
- User Accounts
- Search Functionality
- Payment Gateway Integration
- Mobile Responsiveness
- Admin Section


## Tech Stack

**Client:**  @emotion/react, @emotion/styled, @hookform/resolvers,  @reduxjs/toolkit, react-stripe-js, axios, bootstrap,carousel, react, react-bootstrap, react-dom, react-google-recaptcha, react-hook-form, react-icons, react-redux, react-router-dom, react-scripts, redux-persist, slugify, styled-components,  swiper, web-vitals, 

**Server:** JSON  SERVER
